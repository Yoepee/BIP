## 프로젝트 명 : BIP (Berry Important Promise)

### 프로젝트 설명
- 올바른 약속 문화 만들기
- 상호 합의간에 조건을 걸고 조건을 지키면 서로 패널티가 없으나, 한쪽 상대방이 어기면 패널티를 지급하는 사이트
- 자신과의 약속, 상대방과의 약속, 팀과의 약속으로 구분하여 다양한 약속을 처리하는 기능
